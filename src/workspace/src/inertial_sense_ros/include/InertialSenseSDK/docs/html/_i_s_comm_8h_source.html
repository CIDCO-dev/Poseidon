263 /*
264 Packet Overview
265 
266 Byte
267 0 Packet start byte
268 1 Packet indo: ID (mask 0x1F) | reserved bits (mask 0xE)
269 2 Packet counter (for ACK and retry)
270 3 Packet flags
271 
272 // packet body, may or may not exist depending on packet id - packet body is made up of 4 byte or 8 byte values.
273 4-7 Data identifier
274 8-11 Data length
275 12-15 Data offset
276 16-19 Data start
277 (n-8)-(n-5) Last piece of data
278 // end data
279 
280 n-4 Reserved
281 n-3 Checksum high byte
282 n-2 Checksum low byte
283 n-1 Packet end byte
284 */
